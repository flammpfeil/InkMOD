【MOD】InkMOD-@MC_VERSION@-v@VERSION@

【概要】
バケツでイカ墨がとれるMODです。
イカを倒さずしてイカ墨がとれます。
レシピのための使用が主になります。
飲むと牛乳みたいにポーションの効果がきえます。

【前提MOD】
・MinecraftForge(開発時は @FORGE_VERSION@)

【導入手順】
1.META-INFを消して前提MODを導入
2.このMODをminecraft.jarに直接いれるかmodsフォルダにzipファイルのままいれる
3.完了
